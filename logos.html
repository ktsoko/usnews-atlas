It is a copyright protected trademark and can only be used with permission of authorized representativves of the U.S. News Media Group.

Which file should I use?

  • JPG files can be used for display on the web or presentations. It comes with a white background, so it should not be placed on top of colored backgrounds or imagery. These particular jpgs are not prepared for print. If you need a JPG for print, please contact the design team.
  • PNG files are saved with transparent backgrounds and are the ideal file type for placing logos on top of photographs or other colored backgrounds. PNG files are low resolution and should not be used for print.
  • EPS files are vector images and can be scaled to any size without quality loss. It is the ideal file type for print materials, but can not be used for display on the web. EPS files also benefit from a transparent background.
  • SVG files are vector images that are supported by web browsers. When possible, this should be the primary file type for logos displayed on our website. SVG files allow us to provide a high quality image on all screen sizes, including retina devices.
