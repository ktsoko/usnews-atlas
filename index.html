Flat Icon Guidelines

You can import and use the Icon Macro described below: svgIcon(iconName, color='')

The generated markup looks like
<svg class="icon"><use xlink:href="static/sprites/svg/sprite.symbol.svg#NAME"/></svg>

It will make a call to an external svg sprite file on static and return the icon that matches the hashmark fragment id appeneded to the url, i.e #NAME above. To see a list of icons we can use, take a look at the Flat Icon Library above.

Note: Internet Explorer doesn't support <use> with an external source at all, so we are running the "svg4everybody" JS tool that inlines external SVG content with ajax in browsers that need it.

A basic svg icon with a class of .icon will default to black and 1.5rem square.
